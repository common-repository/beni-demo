Beni demo is a companion plugin for MobeenRaheem's themes.

== Description ==

This Beni Demo plugin is designed just for MobeenRaheem's themes. This plugin creates necessary pages & serves as a demo importer of images & widgets for MobeenRaheem's themes.


Note: All demo content will be imported under the current user.
